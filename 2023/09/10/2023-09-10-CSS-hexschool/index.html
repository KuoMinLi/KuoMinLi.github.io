六角學院-2023切版直播班

當初為什麼報名這堂課？

於今年轉職成前端後，雖然工作上大部分時間是處理 JS 邏輯、API 串接問題，但每次遇到 CSS 問題都會花得比預期久的時間處理，而且過程中會不斷懷疑這樣寫法是否正確，因此就想打掉重練，再來磨一下切版技能。

第一週~第八週的學習過程

前幾週課程真的提供豐富的打底基礎，讓我都不禁懷疑我的CSS到底是怎麼學得，可以如此沒系統，後面四週導入 Bootstrap5 框架後，體會到框架帶來的好處，一旦環境參數先設定好，爆量快速切版真的是輕易達成！

最大的收獲是？

對於身處在銀行軟體業的工程師，看到的版型永遠千篇一律；能在切版班看到各種有趣的設計稿，花心思切好完成時，成就感真的超讚的啦！

最喜歡直播班的哪些活動？

透過小組討論，可以分享每個人切版想法，也剛好我們這組大家都是有經驗的工程師，後來都變成訴苦水的小組討論XD

身為學長姐，分享些想入坑的新同學一些勉勵的話

切版真的是硬底子的基礎，做任何專案都可能會碰到！當你看到 CSS Bug、畫面跑版時，你會覺得頭痛不想解，就代表你需要來參加切版直播班！
參加後，看到問題後只會有我當初怎麼會這樣寫的想法～
